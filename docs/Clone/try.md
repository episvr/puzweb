# 你尝试过了吗？

## 1. 对于新手

<div class="grid cards" markdown>

- 看看标题和flavortext？
- 百度（必应，谷歌）一下？
- 尝试每个看起来不对劲的想法？
- 看看常用工具网站列表？
- 看看网页上任何有用的东西、资料、信息

</div>

## 2. 字母提取

<div class="grid cards" markdown>

- 按字典序排序？
- 提取首字母？
- 试试anagram？
- 观察字母出现的频率？
- 试试凯撒？
- 字母转数字？
- 当成氨基酸序列？
- 当成cryptic解？
- 在什么歌里出现过吗？
- 是缩写或短语吗？
- 提取对角线？
- 看看键盘？
- 双关？

</div>

## 3. 数字提取

<div class="grid cards" markdown>

- 在坐标系里描点？
- 统计出现频率？
- 检查数字的进制转换？
- 数字转字母？
- 当成电话号码？
- 当成ip地址？
- 当成日期？
- 当成原子序数？
- 当成经纬坐标？
- 看看有没有奇怪的序列？
- 看看质数？
- 看看基本常数？
- 想想有没有其他数字和这些有相似之处？

</div>

## 4. 换个方向看看问题

<div class="grid cards" markdown>

- 从镜子里看？
- 从远处看？
- 倾斜着看？
- 倒着看？
- 穿过它看？
- 工整地重新写一遍？（推荐等宽字体）
- 在方格纸上抄一遍？
- 念给另一个人听？
- 从出题人的角度想想？
- 把它变成立体的？
- 裁开来？
- 折起来？
- 把点连起来？

</div>

## 5. 这是什么？

<div class="grid cards" markdown>

- 歌，诗，书，电影，电视剧？
- 盲文？
- 摩斯电码？
- 现代文化？
- 出题人的出题风格？
- 涂鸦？

</div>

## 6. 开摆！

<div class="grid cards" markdown>

- 相信你的直觉？
- 相信别人的直觉？
- 问问“这有哪里很奇怪吗？”
- 返回上一步看看？
- 整理一下思绪？
- 检查一下进度？
- 让别人帮你检查进度？
- 向别人讲解你的进度？
- 把你的进度重新做一遍？
- 问问这方面的专家？
- 出去走走休息一下？
- 问问别人认识这个东西吗？
- 想想还有啥没用过的？
- 坚持一些看起来不对劲的思路？
- 暴力破解？
- 换个题做做？
- 把任务拆分一下分组合作？
- 问问解谜大神？
- 问问解谜新人？
- 想想“有什么特殊的格式吗”？
- 重新读一遍题？

</div>
