# Puzzlehunt in 5 min

> 基本上本文是上一篇介绍的浓缩版

## 它是什么？

**谜题寻宝是一种团队解谜游戏。** 但更准确地说：你面对一堆没有明确指示的谜题，每个谜题的最终目标都是找出一个 **答案**（通常是一个单词或短语）。

提交答案，正确后会解锁更多谜题，最终解开终极谜题赢得比赛。

## 谜题长什么样？

**任何东西都可能是谜题。**

一张图、一段文字、一首歌、一个可执行文件、一堆数据……没有固定的形式。关键特征是：

- 没有“该怎么做”的说明（自己琢磨是解谜的一部分）
- 所有信息最终都会“收束”成一个答案

## 怎么开始解？

1. **相信第一直觉** — 如果看到数独，先当数独解；看到图片，先看图上有什么
2. **检查首字母** — 这是最常用的技巧，永远值得一试
3. **留意数字** — 数字常常是索引，告诉你取某个单词的第几个字母
4. **找共同点** — 多个答案之间可能有某种规律（都以A-F开头？都是某本书的内容？）
5. **提取** — 当你得到一堆信息后，需要通过索引、排序、编码等方式把它压缩成一个简短答案

## 元谜题

当解完一堆普通谜题后，你会遇到 **元谜题** — 它是“关于谜题的谜题”，把普通谜题的答案组合起来，产生最终答案。

解元谜题的第一步通常是：**找到所有答案的共同属性**。

## 常用工具

- **OneLook / Nutrimatic** — 用通配符搜索单词和短语
- **Qat** — 复杂模式匹配
- 搜索引擎
- 大语言模型

**最重要的建议**：几乎所有答案都会以某种方式“收束”，如果你发现自己得到了一堆看似无关的字母或单词，停下来想想——它们之间有什么规律？应该按什么顺序排列？

现在，去找个谜题试试吧！
