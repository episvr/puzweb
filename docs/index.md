# Welcome

## Puzzles Offer Pleasure!

本网站的建立初衷有二：

1. **资源库建设：** 希望创建一个便捷的中央平台，汇集所有与解谜活动相关的资源，帮助新手和资深玩家更好地参与其中。
2. **博客迁移：** 把自己的一些与解谜活动相关的博客迁移到这个平台，与大家分享我的见解、技巧和经历。

这个网站离不开各位解谜同好的贡献（特别是群里丰富的资料和一直在维护更新的共享表格），本页面的设计就来自于 [Wonderepublic 老师维护的新人导航](https://docs.qq.com/sheet/DTnNQb2ZOa0RsdGpI)。

最后，希望来到这个页面的你今天也依然享受解谜，热爱生活。

> 如果有想添加补充（或者删改）的资料欢迎联系我！
