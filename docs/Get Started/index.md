# Puzzle Hunt 快速入门

Puzzle Hunt 是一种以解谜为核心的活动，参与者通过解决一系列谜题来获取线索，最终达到特定的目标。这种活动通常强调团队合作和创造力，适合喜欢挑战智力和逻辑思维的人。

Puzzle(狭义) 是一种谜题类型，目标是通过谜面得到一个特别的答案。

## 术语解释

这里解释了一些你可能想要了解的“黑话”！

- **MITMH**：全称 MIT Mystery Hunt。全球范围内历史最悠久，规模最大，难度最高的 puzzle hunt 活动，每年一月在麻省理工学院校内举行，亦可线上参赛。

- **GPH**：全称 Galactic Puzzle Hunt。是由 ✈✈✈Galactic Trendsetters✈✈✈ 这支解谜队伍举办的线上 puzzle hunt 活动，每年一次，以创新度高著称。

- **Huntinality/Hunti**：由解谜队伍 Cardinality 举办的线上 puzzle hunt 活动，每年一次，结构比较传统，难度适中。

- **CCBC**：全称 Cipher & Code Breaking Competition。中文环境历史最悠久的解谜活动，由密码菌主办。在每年 8 月举办。

- **P&KU**：全称 Puzzle & Key Universe。中文环境新锐的 puzzle hunt 活动，主办者是 winfrid。

- **meta**：全称 metapuzzle，指 puzzle hunt 里一类特殊的谜题，解开它们需要用到其他谜题的相关信息，如题干和答案等。

- **final meta/fm**：通常指一次 puzzle hunt 活动中的最终需要解决的谜题，一般会用到所有的小题信息。

- **enumeration**：用来表明单词或短语长度的数字或其他标识，比如(6 4)就表明这里需要的是一个长度为 6 和 4 的单词构成的短语，例如 finish line。

- **index**：用于从已知信息中提取新的信息的数字或其他标识，比如 finish(3/6)，6 为 enumeration，3 为 index，表示我们需要提取第三个字母 n。

- **backsolve/反解**：与正解(forward solve)相对应。指通过 meta puzzle 等其他途径推断小题答案可能的形式与结构进而得到答案，而非正向解开某道谜题。

- **subpuzzle/sub**：一道谜题中的若干小谜题。

- **cluephrase**：指作为一道谜题中作为中间结果的语句，通常需要对其进行转换得到答案。如 first s in the ussr 是一句 cluephrase，答案为 soviet。

- **flavortext/ft**：一道谜题在其正文部分以外的文本，通常会暗示这道题的解题路径，有时也会起到推进剧情或给谜题增添风味的作用。

- **crossword/cw**：填字游戏。一类在指定的区域内根据描述填入对应单词的文字类谜题，最常见的形式是横纵两个方向交织的方框。

- **cryptic**：一类特定的文字游戏，其描述句通常包含单词的定义和字母组合。

- **meta match**：一类谜题类型，解谜者需要将给出的答案分组，将每个组别视为一个 subpuzzle 进行解题，最后综合起来提取的谜题，如 C13 的《提交记录》。

- **conundrum**：一种谜题类型，其特征是需要解谜者依照文本要求完成其指定的任务来获得答案，通常文本量巨大。

- **mangled clue**：一种谜题类型，作为线索的每个语句中都使用了一些 wordplay，需要对答案也进行相同的变换。如 C11 的第 16 题。

- **physical puzzle**：实体谜题，这类谜题通常需要解谜者使用一些实体的道具或者前往现实中的场所获得线索。

- **纸笔**：纸笔谜题的简称。一类通常以数字、线段和图形为线索的逻辑推理谜题，比较著名的有数独、数织、数回等。

- **日谜**：指日本人制作的谜题，也指代具有类似风格的谜题。其特点是篇幅短小，设计精巧，通常会利用文字上的或者现实生活中的常识与巧合。

- **nutrimatic/nutri**：一个常用的单词/词组匹配搜索的网站，其网址是 [nutrimatic.org](https://nutrimatic.org/)。

- **qat**：一个常用的单词搜索网站，其网址是 [quinapalus.com/qat.html](https://www.quinapalus.com/qat.html)。

- **CPA**：全称 China Puzzle Awards，一个鼓励新人谜题作者制作发表谜题的活动。

- **TPA**: 全称 The Puzzle Awards， 一个在解谜爱好者为谜题活动、谜题制作者/团体颁奖的活动，在每年年终举办。

- **anagram/ana**：指字母重排，如 tea 和 eat。

- **milestone/里程碑**：指一道题的解题过程中得到的中间结果，通常指示了下一步需要进行的操作。

- **brute force/爆破**：指在未得到答案的全部信息的情况下通过暴力枚举可能的字母组合来尝试得到答案的操作。

- **call back**：指某道谜题用到了先前某些谜题（也可是非同一次活动中）的相关信息如题干，但往往不会用到其答案。与 meta 有所区别。

- **单表替换/单表**：指用另一套符号系统逐个对应字母表每个字母的操作。

- **字典序**：指将线索答案按字典条目排列顺序进行设置，常常可以用来辅助推断和检验一些线索的答案，同时也常常暗示需要对线索进行重新排序。

- **red herring/红鲱鱼**：指误导性的或者错误的线索。
